## Featured Projects

### [evalkit](https://github.com/cortexark/evalkit) -- LLM Evaluation Framework

[![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)](https://github.com/cortexark/evalkit)
[![OpenAI](https://img.shields.io/badge/OpenAI-412991?style=flat&logo=openai&logoColor=white)](https://github.com/cortexark/evalkit)
[![Anthropic](https://img.shields.io/badge/Anthropic-191919?style=flat&logo=anthropic&logoColor=white)](https://github.com/cortexark/evalkit)
[![DuckDB](https://img.shields.io/badge/DuckDB-FFF000?style=flat&logo=duckdb&logoColor=black)](https://github.com/cortexark/evalkit)
[![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=flat&logo=streamlit&logoColor=white)](https://github.com/cortexark/evalkit)

A production-grade evaluation framework for LLM applications. Supports LLM-as-Judge scoring, statistical analysis of model outputs, regression detection across prompt iterations, and interactive dashboards for evaluation results. Built to answer the question every AI team asks: *"How do we know this model change is actually better?"*

**Key capabilities:** multi-model evaluation pipelines, configurable judge criteria, DuckDB-backed analytics, Streamlit visualization layer, CI/CD integration for automated regression checks.

### [devx-ai](https://github.com/cortexark/devx-ai) -- AI Developer Tooling Platform

[![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)](https://github.com/cortexark/devx-ai)
[![FastAPI](https://img.shields.io/badge/FastAPI-009688?style=flat&logo=fastapi&logoColor=white)](https://github.com/cortexark/devx-ai)
[![Tree-sitter](https://img.shields.io/badge/Tree--sitter-6DB33F?style=flat&logoColor=white)](https://github.com/cortexark/devx-ai)

An intelligent developer platform that combines code analysis with LLMs to automate code review, generate context-aware documentation, and surface codebase insights. Uses tree-sitter for language-agnostic AST parsing and FastAPI for service orchestration.

**Key capabilities:** automated PR review with LLM-powered analysis, codebase knowledge graph construction, intelligent code search, GitHub integration for seamless developer workflows.
